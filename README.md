# Calculator

## Overview

- An IOS theme calculator
- Sidebar on the side for navigation
- Navigation tab on the top of the page

## Features

1. Standard math functions:

- Basic math: +, -, x, รท
- Scientific: %, !, ^
- Programmer: hex, dec, oct, bin

2. Converter:

- Currency
- Volume
- Length
- Weight and Mass
- Temperature
