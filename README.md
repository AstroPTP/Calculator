# Calculator

## Overview

- An IOS theme calculator
- Sidebar on the side for navigation
- Navigation tab on the top of the page

## Features

1. Standard math functions:
   a. Basic math: +, -, x, รท
   b. Scientific: %, !, ^
   c. Programmer: hex, dec, oct, bin
2. Converter:
   a. Currency
   b. Volume
   c. Length
   d. Weight and Mass
   e. Temperature
